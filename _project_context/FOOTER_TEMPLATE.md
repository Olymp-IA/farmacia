# Footer Template - Farmacia SaaS

Este archivo contiene el footer estandar para toda la documentacion del proyecto.

## Uso

Agregar al final de cada archivo .md de documentacion:

```markdown
---

> powered by **OLYMP-IA.cl** | [https://olymp-ia.cl](https://olymp-ia.cl) | [GitHub](https://github.com/Olymp-IA)
```

## Regla

Segun COMMIT_PROTOCOL.md, toda documentacion creada debe incluir este footer.
